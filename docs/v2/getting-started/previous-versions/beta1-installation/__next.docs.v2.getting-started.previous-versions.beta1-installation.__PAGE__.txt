# Yaci Store 2.0.0-beta1 Installation

Yaci Store provides three types of distributions to suit different deployment needs:

## Types of Distribution

### 1. [Zip Distribution (JARs)](/docs/v2/getting-started/installation/beta1/zip)

Standalone JAR files with configuration files. Provides maximum flexibility for custom setups and enterprise deployments.

**Best for:**
- Custom database configurations
- Enterprise deployment pipelines
- Advanced users requiring full control

[View Zip Distribution Guide →](/docs/v2/getting-started/installation/beta1/zip)

---

### 2. [Docker Distribution](/docs/v2/getting-started/installation/beta1/docker) *(Legacy)*

Complete environment with PostgreSQL for beta1. This is the legacy Docker distribution method.

**Best for:**
- Beta1 users
- Legacy deployments

[View Docker Distribution Guide →](/docs/v2/getting-started/installation/beta1/docker)

---

### 3. [Native (GraalVM) Distribution](/docs/v2/getting-started/installation/beta1/native) *(Preview)*

Native binaries compiled with GraalVM for optimal performance and reduced resource usage.

**Best for:**
- Performance-critical deployments
- Resource-constrained environments
- Testing native compilation capabilities
