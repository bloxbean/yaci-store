# 📦 What's Yaci Store?

**Yaci Store** is a high-performance, production-ready Cardano blockchain indexer built for developers who need reliable access to comprehensive blockchain data. Trusted by the **Cardano Foundation** for their official Rosetta API, Yaci Store combines enterprise-grade reliability with developer-friendly modularity.

## 🚀 Key Capabilities

- **🏛️ Independent Ledger State**: Calculate epoch stakes, rewards, and governance data directly from on-chain data
- **📊 Extended Network Support**: Mainnet (epoch 569), Preprod (epoch 227), Preview (epoch 989)
- **🔧 Modular Architecture**: Use only the components you need with Spring Boot integration
- **📈 Production Monitoring**: Built-in Prometheus metrics and Grafana dashboards
- **⚡ High Performance**: Java 21 virtual threads for optimal throughput

## 🎯 Core Features

**Yaci Store** empowers developers to build sophisticated Cardano applications with:

- **🔍 Comprehensive Data Access**: Blocks, transactions, UTXOs, assets, metadata, and governance data
- **🎛️ Selective Indexing**: Choose exactly what data to index for optimal performance
- **📐 Custom Filtering**: Filter data based on addresses, assets, or custom conditions
- **🔄 Rollback Management**: Automatic handling of chain reorganizations
- **🚀 Out-of-the-Box Applications**: Ready-to-use indexers for common use cases
- **🔌 Plugin System**: Extend functionality with JavaScript, Python, or MVEL scripts
