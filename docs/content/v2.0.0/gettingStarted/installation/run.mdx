## ğŸ¯ One-Command Setup

### Prerequisites
- **Java 11** or **Java 17** ([Download here](https://adoptium.net/temurin/releases/))
- **4 GB RAM** minimum

### Run Yaci Store

```bash
# Download and run the quick-start package
curl -L https://github.com/bloxbean/yaci-store/releases/latest/download/yaci-store-quickstart.jar -o yaci-store-quickstart.jar

# Start Yaci Store with sample data
java -jar yaci-store-quickstart.jar --quickstart
```

**ğŸ‰ That's it!** Yaci Store will:
1. Start with embedded H2 database
2. Load sample blockchain data
3. Launch REST API on `http://localhost:8080`
4. Open web UI at `http://localhost:8080/swagger-ui.html`

## ğŸ” What's Included

### Sample Data
- **Latest 1000 blocks** from Cardano mainnet
- **Transaction data** with full metadata
- **UTxO information** for sample addresses
- **Native assets** and token data
- **Staking information** for active pools

### Available APIs
- **Blocks API**: `/api/v1/blocks`
- **Transactions API**: `/api/v1/transactions`
- **Addresses API**: `/api/v1/addresses`
- **Assets API**: `/api/v1/assets`
- **Epochs API**: `/api/v1/epochs`

### Example Queries
```bash
# Get latest blocks
curl http://localhost:8080/api/v1/blocks?page=0&size=10

# Get transactions for a specific block
curl http://localhost:8080/api/v1/blocks/{block-hash}/transactions

# Check API health
curl http://localhost:8080/actuator/health
```
