# üìã API Reference

Welcome to the comprehensive REST API documentation for **Yaci Store v0.1.x**. This guide covers all available endpoints across different stores and modules.

## üåê Base Configuration

All API endpoints are served on the configured port (default: `8080`) with the base path structure:

```
http://localhost:8080/api/v1/{store}/{endpoint}
```

## üìä Available Store APIs

### üèóÔ∏è Block Store API

**Base Path**: `/api/v1/blocks`

| Endpoint | Method | Description | Example |
|----------|--------|-------------|---------|
| `/` | GET | Get all blocks | `GET /api/v1/blocks` |
| `/{blockNumber}` | GET | Get block by number | `GET /api/v1/blocks/1234567` |
| `/hash/{blockHash}` | GET | Get block by hash | `GET /api/v1/blocks/hash/abc123...` |
| `/latest` | GET | Get latest block | `GET /api/v1/blocks/latest` |
| `/epoch/{epochNumber}` | GET | Get blocks in epoch | `GET /api/v1/blocks/epoch/350` |

**Sample Response**:
```json
{
  "blockNumber": 1234567,
  "blockHash": "abc123def456...",
  "slot": 98765432,
  "epochNumber": 350,
  "epochSlot": 12345,
  "previousBlockHash": "def456abc123...",
  "issuerVkey": "pool1abc123...",
  "vrfKey": "vrf_vk1abc123...",
  "blockTime": 1640995200,
  "blockSize": 2048,
  "txCount": 15
}
```
