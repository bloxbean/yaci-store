# 📦 What's Yaci Store?

**Yaci Store** is a modular Java library and chain indexer designed for developers building custom Cardano blockchain indexing solutions. Built with a flexible architecture, every component is available both as a standalone Java library and as a Spring Boot starter, allowing developers to choose exactly what they need.

## 🎯 Core Features

**Yaci Store v0.1.x** provides essential indexing capabilities:

- **🔧 Modular Design**: Choose specific components for your indexing needs
- **📊 Selective Indexing**: Index only the data your application requires
- **🎛️ Custom Filtering**: Filter data based on addresses, assets, or custom conditions
- **🔄 Rollback Handling**: Automatic management of chain reorganizations
- **🚀 Ready-to-Use Apps**: Out-of-the-box indexers for common use cases
- **⚙️ Extensible Architecture**: Override default behaviors and add custom logic
