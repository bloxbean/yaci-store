## 📦 What is Yaci Store?

**Yaci Store** is a comprehensive Cardano blockchain indexing solution that works in two ways: as a **ready-to-use indexer** for immediate deployment, and as a **modular Java library** for building custom indexing solutions. Whether you need quick setup or tailored functionality, Yaci Store provides both out-of-the-box applications and developer-friendly libraries with Spring Boot integration.

## What You Can Build
- **Custom Indexers**: Create specialized indexers for specific use cases
- **Analytics Platforms**: Build comprehensive blockchain analytics tools
- **DApp Backends**: Power decentralized applications with reliable data access
- **Integration Services**: Connect traditional systems with Cardano blockchain data
- **Monitoring Tools**: Track network activity and performance metrics

## Core Features
- **Modular Design**: Choose specific components for your indexing needs
- **Spring Boot Integration**: Seamless integration with Spring ecosystem
- **Database Support**: PostgreSQL, MySQL, and H2 backends
- **Real-time Indexing**: Keep up with the latest blockchain data
- **REST API**: Query indexed data via comprehensive REST endpoints
- **Horizontal Scaling**: Deploy across multiple instances
- **Selective Processing**: Index only the data your application requires
