# ⚙️ Requirements

Yaci Store requires a development environment with a modern Java setup, basic build tools, and an optional PostgreSQL database for advanced features. Below are the minimum and recommended hardware and software requirements to help you get started smoothly.


## 🖥️ Hardware Requirements

| Component    | Minimum         | Recommended          |
|--------------|------------------|-----------------------|
| **CPU**      | 2 cores          | 4 cores               |
| **RAM**      | 4 GB             | 8 GB                  |
| **Disk**     | 10 GB (any disk) | SSD (for performance) |
| **Network**  | Stable internet connection for updates and package installations |

---

## 💽 Software Requirements

### 🧑‍💻 Operating System

- ✅ **Linux**: Ubuntu 20.04+ (recommended)  
- ✅ **macOS**: 10.15+ (Catalina or later)  
- ✅ **Windows**: 10+ with **WSL2** for Linux compatibility

---

### ☕ Java Environment

- **OpenJDK 11** or later

---

### 🛠️ Build Tools

- **Maven**: `>= 3.6`  
- **Git**: For version control

---

### 💻 Development Environment

- **IDE**:  
  - IntelliJ IDEA  
  - Eclipse  
  - Any Java-compatible IDE  
- **Docker** *(optional)*: For containerized deployments  
- **PostgreSQL** *(optional)*: Version 12+ for advanced configurations

---

### 🌐 Frontend / Auxiliary Dependencies

| Tool        | Required For            | Version       |
|-------------|--------------------------|----------------|
| **Node.js** | Frontend development     | `>= 14`       |
| **NPM**     | Managing JS packages     | `>= 6`        |

---

## 🧰 Optional Tools

- 🧪 **Postman**: API testing GUI  
- 🔗 **cURL**: Command-line HTTP requests  
- 📘 **Swagger UI**: Interactive API documentation & testing
