# Yaci Store Documentation

Welcome to the comprehensive documentation for Yaci Store, a powerful Cardano blockchain data indexing and storage solution.

## What is Yaci Store?

Yaci Store is a modular, high-performance indexing solution for the Cardano blockchain. It allows you to:

- **Index blockchain data** efficiently and reliably
- **Store and query** transaction data, UTXOs, metadata, and more
- **Build custom applications** using blockchain data
- **Scale horizontally** with configurable storage backends

- **Modular Architecture**: Use only the components you need
- **Spring Boot Integration**: Seamless integration with Spring ecosystem
- **Multiple Storage Backends**: PostgreSQL, H2, and more
- **Real-time Indexing**: Keep up with the latest blockchain data
- **REST API**: Query indexed data via comprehensive REST endpoints
- **Horizontal Scaling**: Deploy across multiple instances
